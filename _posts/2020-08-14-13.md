# input 파일 만들기

C 파일을 생성하고
clang name.c -o name 명령어를 통해 컴파일

clang -O3 -emit-llvm name.c -c -o name.bc 명령어를 통해 c파일을 llvm 비트코드 파일로 변환

./name 또는 lli name.bc를 통해 실행할 수 있음

gcc hello.s -o hello.native
